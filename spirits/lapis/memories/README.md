# lapis memories

this directory is layered on purpose.

- `long-term.md` is the compact top-of-head memory
- `window/` is the rolling recent-memory window
- `archive/` is for lower-signal memory residue that should persist without staying prompt-attached
- everything else here is durable searchable storage

do not treat this directory like a scratchpad.

the daily thread is for live work.
artifacts are for outputs.
memory should stay compact enough to be worth loading.
